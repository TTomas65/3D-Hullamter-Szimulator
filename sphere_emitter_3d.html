🎯 3x3-as Térrács Kiválasztás

Utasítások:
1. A 3D térben megjelent egy 3×3×3-as kockaháló 27 ponttal.
2. 1× kattintás → ● Sárga (vízelem) = álló forráspont
3. 2× kattintás → ● Piros (tűzelem) = mozgó forráspont (felugró ablak a sebesség és irány beállításához)
4. 3× kattintás → ● Szürke = visszavonás
5. Jobb klikk piros ponton → tűzelem paramétereinek szerkesztése
6. Ha kész, nyomd meg az Indítás gombot!
Kiválasztott pontok: 0 / 27

Információs Panel:

🖱️ Egér

Bal egérgomb: Kiáradási pont mozgatása
Jobb egérgomb: Kamera forgatása (orbit)
Egérgörgő: Nagyítás / kicsinyítés

⌨️ Billentyűk

W / S: Előre / Hátra
A / D vagy ◀ / ▶: Balra / Jobbra
▲ / ▼: Felfelé / Lefelé
X: Forgatás X tengely körül
Y: Forgatás Y tengely körül
P: Szünet / Folytatás
O: Sebesség szabályozás (1x → 1/2x → 1/4x → 1/8x → 1x)
ő / ú: Kamera forgatása balra / jobbra
á / ű: Kamera forgatása fel / le

📐 Metszési vezérlés

Metszet X gomb: Y‑irányú metszősík ki-/bekapcsolása
Metszet Y gomb: X‑irányú metszősík ki-/bekapcsolása
Metszet pozíció csúszkák: A sík eltolása ± irányban

⚙️ Egyéb funkciók

Intenzitás: Kiáradási sebesség
Átlátszóság: Gömb megjelenítés módja
Mozgástípus: Forráspont animáció módja
Szünet / Újraindítás gomb: Animáció megállítása / folytatása
Kamera reset: Kamera vissza alaphelyzetbe
Képernyőmentés: Az aktuális nézet (csak a tartalom) mentése JPG formátumban panelek nélkül.
Képernyővideó rögzítése: Az aktuális nézet videófelvétele WebM formátumban.
Időszál megjelenítése: A forráspont által kibocsátott gömbök középpontjainak megjelölése és összekötése sárga színnel.

⚙️ Funkciók

Intenzitás: Kiáradási sebesség szabályozása
Átlátszóság: Gömbök megjelenítése (1=drótváz, 0=szilárd)
Forráspont sebesség: Mozgás sebessége X/Y tengely
Mozgástípusok: Különböző animációs módok
